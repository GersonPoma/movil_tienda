# Tienda Móvil - Kreativ Flow 📱

Una aplicación móvil moderna construida con Flutter para la gestión y visualización de productos en una tienda.

## 🚀 Características

- **Catálogo de Productos**: Navegación fluida por las categorías de productos.
- **Búsqueda Avanzada**: Encuentra productos rápidamente.
- **Carrito de Compras**: Gestión intuitiva de pedidos.
- **Interfaz Premium**: Diseño moderno, limpio y responsivo.

## 🛠️ Tecnologías Utilizadas

- **Framework**: [Flutter](https://flutter.dev/)
- **Lenguaje**: [Dart](https://dart.dev/)
- **Gestión de Estado**: Provider
- **Almacenamiento Local**: Shared Preferences
- **API**: Integración con servicios REST

## 🏁 Cómo empezar

### Requisitos previos

- Flutter SDK (v3.11.4 o superior)
- Dart SDK
- Un emulador de Android/iOS o un dispositivo físico.

### Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/tienda-movil.git
   ```
2. Entra en el directorio del proyecto:
   ```bash
   cd tienda-movil
   ```
3. Instala las dependencias:
   ```bash
   flutter pub get
   ```
4. Ejecuta la aplicación:
   ```bash
   flutter run
   ```
